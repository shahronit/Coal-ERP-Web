# TradeCRM Pro Training Videos

This folder contains self-hosted training videos used by the in-app Help Center.

## Folder Layout

```
public/help/
  videos/
    en/   English MP4 files
    hi/   Hindi MP4 files
  thumbnails/
    training.svg
```

## Replace Placeholder Videos

Keep the same filenames when replacing videos:

- `01-getting-started.mp4`
- `02-login-profile.mp4`
- `03-master-data.mp4`
- `04-purchases.mp4`
- `05-sales.mp4`
- `06-inventory.mp4`
- `07-payments.mp4`
- `08-reports.mp4`
- `09-documents.mp4`
- `10-dashboard.mp4`

Put English narration files in `videos/en/` and Hindi narration files in `videos/hi/`.

## Recommended Video Format

- Format: MP4
- Codec: H.264
- Size: 1280x720
- Audio: AAC
- Suggested length: 2 to 5 minutes
- Suggested max file size: 50 MB per video

## Recording Script

For each topic, record:

1. What the screen is used for.
2. Where to click first.
3. What fields to fill.
4. What the user should check before saving.
5. Where to find help if stuck.

Use simple words. Avoid developer terms like API, database, server, or endpoint.
